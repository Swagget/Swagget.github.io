Wanted to create a tool that allowed non-technical users to be able to gather insights from the data from their company and effectively automate the job of a data analyst.


We created a meta language and framework for performing generalized machine learning on a variety of columns. Allowing for clustering, regression and other techniques to gather insights. This framework allowed for massive scalability.


However we soon realized that each company had incredibly messy data and needed heavy cleaning before we were able to use any of it.
